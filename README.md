# TableConverter

C# 하드코딩 테이블 자동 컨버터.
엑셀 테이블을 읽어서 csv파일과 유니티 클라이언트, C#/C++ 서버 소스코드로 자동변환 합니다.
변환된 csv파일와 소스코드는 필요에 따라 확장하여 사용할 수 있습니다.
